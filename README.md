# Tavla Oyunu Demo
Hobi amaçlı geliştirdiğim online tavla oyunu. Vakit olmadığı için refactor edilememiştir. Buglar ve backendde güvenlik açıkları içerir.
# Teknolojiler
- React js
- Node js
- Express
- Socket.io
